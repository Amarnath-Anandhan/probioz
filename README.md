# PROBIOZ Website
A static multi-page website project built with HTML/CSS/JS.
This is a platform to learn programming with biological examples and use cases for biologists.

## Preview
> Not deployed yet.  
> You can view the UI via screenshots below.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Multi-page learning content
- Project and quiz sections
- Bioinformatics-focused projects
- Python content with biological examples
- Embedded Python code runner for content and projects

## Pages
- Home
- Python
- Project
- Quiz
- About Us
- Feedback
- Contact
- Perl, SQL, and ML (not completed yet)
