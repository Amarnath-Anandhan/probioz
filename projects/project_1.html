1.Sequence Utility Toolkit

Beginner

GOAL: Build one program that can clean, check, analyze, and convert a DNA/RNA sequence.

Rule: In this project you will build a single tool step by step. Each cell contains partial code. You must fill the lines marked as TODO, run the cell, check the output, and then move to the next cell. The final cell contains the full working program.

What this tool will do: It takes a sequence from the user, cleans it, checks whether it is DNA or RNA, calculates GC%, converts between DNA and RNA, finds reverse complement for DNA, and allows saving the result to a file.
